HTTP Client with Retry Policy

var retryPolicy = HttpPolicyExtensions
    .HandleTransientHttpError()
    .WaitAndRetryAsync(3, retryAttempt =>
        TimeSpan.FromSeconds(Math.Pow(2, retryAttempt)));

builder.Services.AddHttpClient<IEtisalatCallbackClient, EtisalatCallbackClient>()
    .AddPolicyHandler(retryPolicy);

Services

Services/TicketMonitorService.cs Service

Background service that monitors the Ticket collection and auto-sends callbacks when tickets close.

public class TicketMonitorService : BackgroundService
{
    protected override async Task ExecuteAsync(CancellationToken stoppingToken)
    {
        while (!stoppingToken.IsCancellationRequested)
        {
            await DiscoverNewTicketsAsync();
            await CheckTicketStatusChangesAsync();
            await Task.Delay(TimeSpan.FromSeconds(_settings.PollingIntervalSeconds));
        }
    }
}

Authentication

Authentication/BasicAuthenticationHandler.cs Security

Custom authentication handler for Basic Auth with Base64 encoded credentials.

Authentication Flow

Request -> Extract Auth Header -> Decode Base64 -> Validate -> Success/Fail | Format: "Basic {base64(user:pass)}"
Security Note: Enable IP whitelisting in production. Add Etisalat's IP addresses to the whitelist.

User Management

Models/User.cs Model

User entity stored in serviceme_users collection. All fields use serviceme_ prefix.

public class User
{
    [BsonId]
    [BsonRepresentation(BsonType.ObjectId)]

    [BsonElement("serviceme_role")]
