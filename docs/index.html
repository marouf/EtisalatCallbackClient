Complete integration solution for Etisalat XaaS platform with real-time ticket monitoring and automated provisioning callbacks

API Documentation Open Dashboard

Key Features

Enterprise-grade integration with Etisalat XaaS platform

Automated Callbacks

Automatically send ACCEPTED callbacks to Etisalat when tickets are closed. No manual intervention required.

SLA Monitoring

Real-time SLA tracking with configurable rules. Get warnings before violations and auto-expire overdue tickets.

Ticket Tracking

Search by ticket # or subject. Filter by status, callback state, or failed callbacks. Fill missing reference numbers from UI.

Secure Authentication

Basic auth for API, cookie auth for UI. User management with roles (Admin/User), password hashing, and session control.

Retry on Failure

Failed callbacks can be retried from the UI. Automatic Polly retry policies for transient network failures.

Docker Ready

Production-ready Docker configuration with MongoDB, health checks, and easy deployment via docker-compose.

Quick API Reference

POST /api/provisioning Receive provisioning requests from Etisalat
GET /api/monitor/tickets List tracked subscription tickets
POST /api/monitor/callback Send manual callback to Etisalat
GET /api/monitor/sla/violations Get SLA violations and warnings
GET /api/health Health check endpoint
GET /Users User management (Admin only)
